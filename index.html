vdanviel/metablog

Um projeto desenvolvido com a stack MERN, que consiste em uma rede social onde os usuários podem postar conteúdos com mídias, seguir outros usuários e personalizar seus perfis. O projeto adota o conceito de monorepositório, reunindo tanto o lado do servidor quanto o lado do cliente em um único repositório. Além disso, ele está dockerizado, permitindo que o ambiente seja iniciado facilmente com um simples comando docker-compose.

vdanviel/serene-app

Serene é um aplicativo psicológico desenvolvido com o objetivo de explorar e pesquisar o potencial da IA. Criado em React Native, com um servidor em Symfony, o app gera cinco perguntas aleatórias sobre ansiedade. Após o usuário responder ao questionário, o aplicativo fornece um diagnóstico preliminar indicando, com base nos dados fornecidos, se há sinais de ansiedade. As inteligências artificiais utilizadas no projeto foram extraídas da plataforma Hugging Face.
